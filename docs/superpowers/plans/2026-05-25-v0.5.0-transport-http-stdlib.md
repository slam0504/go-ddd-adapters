# v0.5.0 `transport/http/stdlib` + Health Adapter — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship the adapter half of v0.5.0 — a public `transport/http/stdlib` HTTP server adapter with synchronous-bind / graceful-shutdown semantics, plus a `transport/http/stdlib/health` sub-package that wires multiple `ports/health.Check`s into `/healthz` (liveness) and `/readyz` (readiness) endpoints.

**Architecture:** `httpstdlib.New(addr, handler, opts...)` returns a `*Server` whose `Module()` binds the listener synchronously at `StartFn` time (so `Addr()` works for `:0`) before launching `Serve` in a goroutine; `StopFn` runs `server.Shutdown` under a timeout. `health.Registry` aggregates named `corehealth.Check`s; its `Handler()` uses Go 1.22 method-aware mux to expose `GET /healthz` (cheap always-200) and `GET /readyz` (sequential Check execution with per-Check `ProbeTimeout`, 200 all-pass / 503 any-fail, JSON body listing every Check). Liveness intentionally never runs Checks. No middleware framework, no router beyond mux.

**Tech Stack:** Go 1.25, `net/http` stdlib mux (1.22 method routing), `go-ddd-core` pseudo-versioned at commit `53fd5b2` for development (swapped to `v0.5.0` at the tag-step gate).

**Spec:** `docs/superpowers/specs/2026-05-25-v0.5.0-transport-http-stdlib.md` (APPROVED 2026-05-25 with §5.1 synchronous-bind / `Server` constructor + §6.1 panic-test removal revisions applied).

```go
// Package httpstdlib is a thin bootstrap-aware adapter around net/http
// for inbound HTTP transport. See doc.go for an overview.
package httpstdlib

import (
	"context"
	"net"
	"time"

	"github.com/slam0504/go-ddd-core/ports/logger"
)

const (
	defaultReadHeaderTimeout = 5 * time.Second
	defaultShutdownTimeout   = 15 * time.Second
)

// Option configures a Server constructed via New.
type Option func(*config)

type config struct {
	readHeaderTimeout time.Duration
	shutdownTimeout   time.Duration
	logger            logger.Logger
	moduleName        string
	baseContext       func(net.Listener) context.Context
}

func newConfig(addr string) *config {
	return &config{
		readHeaderTimeout: defaultReadHeaderTimeout,
		shutdownTimeout:   defaultShutdownTimeout,
		logger:            nopLogger{},
		moduleName:        "http:" + addr,
	}
}

// WithReadHeaderTimeout overrides the default 5s read-header timeout.
func WithReadHeaderTimeout(d time.Duration) Option {
	return func(c *config) { c.readHeaderTimeout = d }
}

// WithShutdownTimeout overrides the default 15s shutdown timeout.
func WithShutdownTimeout(d time.Duration) Option {
	return func(c *config) { c.shutdownTimeout = d }
}

// WithLogger replaces the no-op logger with l. Used for start /
// serve-error / shutdown events.
func WithLogger(l logger.Logger) Option {
	return func(c *config) {
		if l != nil {
			c.logger = l
		}
	}
}

// WithModuleName overrides the default "http:<addr>" module name.
func WithModuleName(name string) Option {
	return func(c *config) {
		if name != "" {
			c.moduleName = name
		}
	}
}

// WithBaseContext sets http.Server.BaseContext. Default is nil
// (stdlib uses context.Background per-connection).
func WithBaseContext(fn func(net.Listener) context.Context) Option {
	return func(c *config) { c.baseContext = fn }
}

// nopLogger is the zero-config default. It satisfies logger.Logger
// without doing any work.
type nopLogger struct{}

func (nopLogger) Log(context.Context, logger.Level, string, ...logger.Attr) {}
func (nopLogger) With(...logger.Attr) logger.Logger                         { return nopLogger{} }
func (nopLogger) Enabled(context.Context, logger.Level) bool                { return false }
```

```go
package httpstdlib

import (
	"context"
	"errors"
	"net"
	"net/http"
	"sync"

	"github.com/slam0504/go-ddd-core/bootstrap"
	"github.com/slam0504/go-ddd-core/ports/logger"
)

// Server wraps an *http.Server with a pre-bound listener so the actual
// listening address (post net.Listen) is exposed via Addr.
type Server struct {
	configuredAddr string
	handler        http.Handler
	cfg            *config

	mu       sync.Mutex
	listener net.Listener
	server   *http.Server
}

// New constructs a Server. The listener is created during Start
// (not at New time) so bind failures surface via the bootstrap
// lifecycle, not the constructor.
func New(addr string, handler http.Handler, opts ...Option) *Server {
	cfg := newConfig(addr)
	for _, opt := range opts {
		opt(cfg)
	}
	return &Server{
		configuredAddr: addr,
		handler:        handler,
		cfg:            cfg,
	}
}

// Addr returns the actual listening address; valid only after Start
// has returned without error. Returns "" before Start or after a
// failed Start.
func (s *Server) Addr() string {
	s.mu.Lock()
	defer s.mu.Unlock()
	if s.listener == nil {
		return ""
	}
	return s.listener.Addr().String()
}

// Module adapts the Server into a bootstrap.Module.
func (s *Server) Module() bootstrap.ModuleFunc {
	return bootstrap.ModuleFunc{
		ModuleName: s.cfg.moduleName,
		StartFn:    s.start,
		StopFn:     s.stop,
	}
}

func (s *Server) start(ctx context.Context, _ *bootstrap.App) error {
	ln, err := net.Listen("tcp", s.configuredAddr)
	if err != nil {
		return err
	}

	srv := &http.Server{
		Handler:           s.handler,
		ReadHeaderTimeout: s.cfg.readHeaderTimeout,
		BaseContext:       s.cfg.baseContext,
	}

	s.mu.Lock()
	s.listener = ln
	s.server = srv
	s.mu.Unlock()

	s.cfg.logger.Log(ctx, logger.LevelInfo, "http listening",
		logger.F("addr", ln.Addr().String()),
		logger.F("module", s.cfg.moduleName))

	go func() {
		if err := srv.Serve(ln); err != nil && !errors.Is(err, http.ErrServerClosed) {
			s.cfg.logger.Log(ctx, logger.LevelError, "http serve error",
				logger.F("err", err.Error()),
				logger.F("module", s.cfg.moduleName))
		}
	}()

	return nil
}

func (s *Server) stop(ctx context.Context) error {
	s.mu.Lock()
	srv := s.server
	s.mu.Unlock()
	if srv == nil {
		return nil
	}

	shutdownCtx, cancel := context.WithTimeout(ctx, s.cfg.shutdownTimeout)
	defer cancel()
	return srv.Shutdown(shutdownCtx)
}
```

```go
func TestServer_StopGracefulShutdown(t *testing.T) {
	released := make(chan struct{})
	mux := http.NewServeMux()
	mux.HandleFunc("GET /slow", func(w http.ResponseWriter, r *http.Request) {
		select {
		case <-released:
			_, _ = io.WriteString(w, "done")
		case <-r.Context().Done():
		}
	})

	srv := httpstdlib.New(":0", mux)
	mod := srv.Module()
	ctx, cancel := context.WithTimeout(context.Background(), 5*time.Second)
	defer cancel()
	if err := mod.Start(ctx, nil); err != nil {
		t.Fatalf("Start: %v", err)
	}

	addr := srv.Addr()

	// Fire an in-flight request; it blocks until we close `released`.
	type result struct {
		status int
		body   string
		err    error
	}
	resCh := make(chan result, 1)
	go func() {
		resp, err := http.Get("http://" + addr + "/slow")
		if err != nil {
			resCh <- result{err: err}
			return
		}
		defer resp.Body.Close()
		b, _ := io.ReadAll(resp.Body)
		resCh <- result{status: resp.StatusCode, body: string(b)}
	}()

	// Give the request time to enter the handler.
	time.Sleep(50 * time.Millisecond)

	// Kick off Stop in another goroutine; it must wait for the in-flight req.
	stopErrCh := make(chan error, 1)
	go func() {
		stopCtx, stopCancel := context.WithTimeout(context.Background(), 5*time.Second)
		defer stopCancel()
		stopErrCh <- mod.Stop(stopCtx)
	}()

	// Release the handler; Stop should return shortly after.
	close(released)

	select {
	case err := <-stopErrCh:
		if err != nil {
			t.Fatalf("Stop returned error: %v", err)
		}
	case <-time.After(3 * time.Second):
		t.Fatalf("Stop did not return within 3s after handler completed")
	}

	res := <-resCh
	if res.err != nil {
		t.Fatalf("in-flight GET: %v", res.err)
	}
	if res.status != http.StatusOK || res.body != "done" {
		t.Fatalf("in-flight result: status=%d body=%q", res.status, res.body)
	}
}
```

```go
// Package health aggregates ports/health.Check probes for the
// transport/http/stdlib adapter. See doc.go for endpoint semantics.
package health

import (
	"fmt"
	"sync"
	"time"

	corehealth "github.com/slam0504/go-ddd-core/ports/health"
)

const defaultProbeTimeout = 2 * time.Second

// Registry aggregates named health.Checks. The zero value is ready
// to use. Register / MustRegister are safe for concurrent calls during
// startup, but adding Checks after the handler is serving requests is
// not recommended.
type Registry struct {
	mu           sync.RWMutex
	checks       []corehealth.Check
	names        map[string]struct{}
	probeTimeout time.Duration
}

// Register adds c to the registry. Names must be unique; a duplicate
// returns an error so wiring fails loud at startup.
func (r *Registry) Register(c corehealth.Check) error {
	r.mu.Lock()
	defer r.mu.Unlock()
	if r.names == nil {
		r.names = make(map[string]struct{})
	}
	name := c.Name()
	if _, dup := r.names[name]; dup {
		return fmt.Errorf("health: duplicate check name %q", name)
	}
	r.names[name] = struct{}{}
	r.checks = append(r.checks, c)
	return nil
}

// MustRegister panics on duplicate names. Convenience for startup wiring
// where a duplicate is a programming error.
func (r *Registry) MustRegister(c corehealth.Check) {
	if err := r.Register(c); err != nil {
		panic(err)
	}
}

// SetProbeTimeout sets the per-Check ctx timeout used during readiness
// probes. A zero or negative value disables the timeout. Default 2s.
func (r *Registry) SetProbeTimeout(d time.Duration) {
	r.mu.Lock()
	defer r.mu.Unlock()
	r.probeTimeout = d
}

// effectiveProbeTimeout returns the configured timeout or the default.
func (r *Registry) effectiveProbeTimeout() time.Duration {
	r.mu.RLock()
	defer r.mu.RUnlock()
	if r.probeTimeout == 0 {
		return defaultProbeTimeout
	}
	return r.probeTimeout
}

// snapshot returns a copy of the registered Checks for handler use,
// so handlers do not hold the registry lock during I/O.
func (r *Registry) snapshot() []corehealth.Check {
	r.mu.RLock()
	defer r.mu.RUnlock()
	out := make([]corehealth.Check, len(r.checks))
	copy(out, r.checks)
	return out
}
```

```go
package health

import (
	"context"
	"encoding/json"
	"net/http"
)

type probeResult struct {
	Name  string `json:"name"`
	OK    bool   `json:"ok"`
	Error string `json:"error,omitempty"`
}

type body struct {
	Status string         `json:"status"`
	Checks []probeResult  `json:"checks,omitempty"`
}

// LivenessHandler serves only liveness semantics: always 200, never runs
// registered Checks. Liveness signals "process can serve HTTP" — a
// failing dependency must not cause an orchestrator to restart the pod.
func (r *Registry) LivenessHandler() http.Handler {
	return http.HandlerFunc(func(w http.ResponseWriter, _ *http.Request) {
		writeJSON(w, http.StatusOK, body{Status: "ok"})
	})
}

// ReadinessHandler serves only readiness semantics: runs every
// registered Check sequentially under the configured ProbeTimeout.
// 200 if all pass, 503 if any fail; the body always lists every Check.
func (r *Registry) ReadinessHandler() http.Handler {
	return http.HandlerFunc(func(w http.ResponseWriter, req *http.Request) {
		checks := r.snapshot()
		timeout := r.effectiveProbeTimeout()

		results := make([]probeResult, 0, len(checks))
		allOK := true
		for _, c := range checks {
			ctx := req.Context()
			var cancel context.CancelFunc
			if timeout > 0 {
				ctx, cancel = context.WithTimeout(ctx, timeout)
			}
			err := c.Check(ctx)
			if cancel != nil {
				cancel()
			}
			res := probeResult{Name: c.Name(), OK: err == nil}
			if err != nil {
				res.Error = err.Error()
				allOK = false
			}
			results = append(results, res)
		}

		status := http.StatusOK
		summary := "ok"
		if !allOK {
			status = http.StatusServiceUnavailable
			summary = "unhealthy"
		}
		// Always emit a checks array (possibly empty) on the readiness body
		// so operators can rely on a stable shape. The 'omitempty' on the
		// type tag is overridden here by emitting a non-nil slice.
		out := body{Status: summary, Checks: results}
		if out.Checks == nil {
			out.Checks = []probeResult{}
		}
		writeJSON(w, status, out)
	})
}

// Handler combines /healthz and /readyz on a method-aware mux. Mount
// at root or via http.StripPrefix.
func (r *Registry) Handler() http.Handler {
	mux := http.NewServeMux()
	mux.Handle("GET /healthz", r.LivenessHandler())
	mux.Handle("GET /readyz", r.ReadinessHandler())
	return mux
}

func writeJSON(w http.ResponseWriter, status int, payload any) {
	w.Header().Set("Content-Type", "application/json")
	w.WriteHeader(status)
	_ = json.NewEncoder(w).Encode(payload)
}
```

```go
	reg.MustRegister(corehealth.NewCheck("always", func(_ context.Context) error { return nil }))
	reg.MustRegister(corehealth.NewCheck("flaky", func(_ context.Context) error {
		if flakyFail.Load() {
			return errAdminFlake
		}
		return nil
	}))

	mainSrv := httpstdlib.New(":0", mainMux)
	adminSrv := httpstdlib.New(":0", reg.Handler(),
		httpstdlib.WithModuleName("admin"))

	app := bootstrap.New(bootstrap.Options{})
	app.Use(mainSrv.Module(), adminSrv.Module())

	ctx, cancel := context.WithTimeout(context.Background(), 10*time.Second)
	defer cancel()

	if err := app.Start(ctx); err != nil {
		t.Fatalf("app.Start: %v", err)
	}
	t.Cleanup(func() {
		stopCtx, stopCancel := context.WithTimeout(context.Background(), 5*time.Second)
		defer stopCancel()
		if err := app.Stop(stopCtx); err != nil {
			t.Fatalf("app.Stop: %v", err)
		}
	})

	// Verify main server.
	resp, err := http.Get("http://" + mainSrv.Addr() + "/ping")
	if err != nil {
		t.Fatalf("main GET /ping: %v", err)
	}
	resp.Body.Close()
	if resp.StatusCode != 200 {
		t.Fatalf("main /ping status = %d, want 200", resp.StatusCode)
	}

	adminBase := "http://" + adminSrv.Addr()

	// Verify /healthz always 200.
	mustStatus(t, adminBase+"/healthz", 200)

	// Verify /readyz starts at 200.
	mustStatus(t, adminBase+"/readyz", 200)

	// Flip the flaky Check; /readyz must transition to 503.
	flakyFail.Store(true)
	mustStatus(t, adminBase+"/readyz", 503)

	// /healthz must remain 200 throughout (liveness ignores deps).
	mustStatus(t, adminBase+"/healthz", 200)

	// Flip back; /readyz returns to 200.
	flakyFail.Store(false)
	mustStatus(t, adminBase+"/readyz", 200)

	// Body shape sanity on a 503 response.
	flakyFail.Store(true)
	body := mustGetJSON(t, adminBase+"/readyz")
	if body["status"] != "unhealthy" {
		t.Fatalf("status = %v, want 'unhealthy'", body["status"])
	}
	checks, _ := body["checks"].([]any)
	if len(checks) != 2 {
		t.Fatalf("len(checks) = %d, want 2", len(checks))
	}
}

var errAdminFlake = &flakeError{}

type flakeError struct{}

func (*flakeError) Error() string { return "flake: simulated failure" }

func mustStatus(t *testing.T, url string, want int) {
	t.Helper()
	resp, err := http.Get(url)
	if err != nil {
		t.Fatalf("GET %s: %v", url, err)
	}
	resp.Body.Close()
	if resp.StatusCode != want {
		t.Fatalf("GET %s status = %d, want %d", url, resp.StatusCode, want)
	}
}

func mustGetJSON(t *testing.T, url string) map[string]any {
	t.Helper()
	resp, err := http.Get(url)
	if err != nil {
		t.Fatalf("GET %s: %v", url, err)
	}
	defer resp.Body.Close()
	var out map[string]any
	if err := json.NewDecoder(resp.Body).Decode(&out); err != nil {
		t.Fatalf("decode %s: %v", url, err)
	}
	return out
}
```

```markdown
## v0.5.0 transport/http/stdlib + health (2026-05-25 cycle)

Decisions locked at kickoff (2026-05-25) and confirmed at implementation:

- **Package path**: `transport/http/stdlib`, pkg name `httpstdlib`. The
  `/stdlib` suffix leaves room for `chi/`, `gin/`, `fiber/` siblings
  later without renaming this one.
- **`Server` + synchronous bind**: `New(...)` returns a `*Server`; its
  `Module().Start` performs `net.Listen("tcp", addr)` synchronously
  before launching `Serve` in a goroutine. Bind errors return from
  `Start`. `Addr()` exposes the resolved address (works for `:0`).
- **Convenience `Module(addr, handler, opts...)`** wraps
  `New(...).Module()` for call sites that don't need `Addr()`.
- **Handler panic semantics**: not the adapter's responsibility.
  `net/http` per-connection recover applies. The adapter does not
  install request middleware and does not promise a 500 on panic.
- **Health endpoint semantics (locked)**:
  - `/healthz` always 200, dep-free; runs no Check.
  - `/readyz` runs every Check sequentially with per-Check
    `ProbeTimeout` (default 2s); 200 all-pass, 503 any-fail; body
    lists every Check on both success and failure for stable
    operator scripting; empty registry → 200.
  - Only GET; other methods → 405.
- **Registry duplicate name** → `Register` returns error,
  `MustRegister` panics. Caught at startup wiring time.
- **No shim around core**. If the `ports/health.Check` contract had
  proven insufficient mid-implementation, the cycle would have
  paused and the change would have been pushed to core first. None
  was needed for v0.5.0.
- **`examples/orders/cmd/api` not migrated** in this cycle. The
  example PR for that is a separate follow-up.
- **Dev-time dependency**: core pinned to a pseudo-version at commit
  `53fd5b2` for the duration of the cycle. Swap to `v0.5.0` at the
  tag-step gate after core annotates the tag.
- **Tag-step gate** (4-step sequence): (1) adapter PR merged with
  pseudo-version pin, CI green. (2) core annotates + pushes
  `v0.5.0`. (3) adapter `go.mod` + `examples/orders/go.mod` bumped
  to `v0.5.0`; CI green at bumped tip. (4) adapter annotates +
  pushes its `v0.5.0` tag.
```
